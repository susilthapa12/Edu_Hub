C Variables & Data Types

Variables are used to store data. Each variable has a data type that defines the kind of data it can hold.

1. Variable Declaration

int age;       // integer
float price;    // floating-point number
char grade;     // single character
double pi;      // double-precision floating point

2. Assigning Values

age = 25;
price = 19.99;
grade = 'A';
pi = 3.14159;

3. Declaring & Initializing in One Line

int a = 10;
float b = 5.5;
char c = 'X';

4. Basic Data Types in C

  • int – stores integers (e.g., 0, 1, -5)
  • float – stores decimal numbers (e.g., 3.14)
  • double – stores double-precision decimal numbers
  • char – stores a single character (e.g., 'A')
  • void – represents no value

5. Printing Variables

#include <stdio.h>

int main() {
    int age = 25;
    float price = 19.99;
    char grade = 'A';

    printf("Age: %d\n", age);
    printf("Price: %.2f\n", price);
    printf("Grade: %c\n", grade);

    return 0;
}

6. Format Specifiers

  • %d – integer
  • %f – float
  • %.2f – float with 2 decimal places
  • %c – char
  • %s – string
