AngularJS Directives

Directives are special HTML attributes that start with ng-. They extend HTML functionality:

<ng-app>

Defines the root of an AngularJS application:

<div ng-app="">
  <p>Hello, AngularJS!</p>
</div>

<ng-model>

Binds the value of HTML controls (input, select, textarea) to application data:

<div ng-app="">
  <p>Name: <input type="text" ng-model="name"></p>
  <p>Hello {{name}}!</p>
</div>

<ng-bind>

Displays data from AngularJS variables in HTML:

<div ng-app="">
  <p ng-bind="name"></p>
</div>

<ng-init>

Initializes AngularJS variables:

<div ng-app="" ng-init="firstName='John'">
  <p>Name: <span ng-bind="firstName"></span></p>
</div>

Summary

  • Directives are HTML attributes with ng- prefix.
  • ng-app initializes an AngularJS application.
  • ng-model binds input data to variables.
  • ng-bind displays variable data in HTML.
  • ng-init initializes variables at page load.
