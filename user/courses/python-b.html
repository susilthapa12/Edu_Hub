Python Variables

A variable is a name used to store data in memory. You can store numbers, text, or other types of data.

Key points:

  • You do not need to declare the data type explicitly.
  • Variable names should be meaningful and start with a letter or underscore.
  • Python is case-sensitive: name and Name are different variables.
# Storing different types of data in variables
x = 10          # integer
name = "Alice"  # string
pi = 3.14       # float
is_active = True  # boolean

print(x, name, pi, is_active)

Python Data Types

Python has several built-in data types. Some common ones are:

  • int: Whole numbers (e.g., 5, -20)
  • float: Decimal numbers (e.g., 3.14, -0.5)
  • str: Text (e.g., "Hello Python")
  • bool: True or False values
# Examples of data types
age = 25           # int
height = 5.9       # float
city = "London"    # string
is_student = False # boolean

print(age, height, city, is_student)

Check Data Type

You can check the type of a variable using the type() function:

x = 42
y = "Python"

print(type(x))  # Output: <class 'int'>
print(type(y))  # Output: <class 'str'>

Basic Operations

Python can perform arithmetic, comparison, and logical operations easily.

Arithmetic Operations

a = 10
b = 3

print(a + b)  # Addition: 13
print(a - b)  # Subtraction: 7
print(a * b)  # Multiplication: 30
print(a / b)  # Division: 3.3333
print(a % b)  # Modulus: 1 (remainder)

Comparison Operations

a = 10
b = 3

print(a == b)  # Equal? False
print(a > b) # Greater than? True
print(a < b) # Less than? False
