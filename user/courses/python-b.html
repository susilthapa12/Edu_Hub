Python Variables

A variable is a name used to store data in memory.

Key points:

  • No need to declare type explicitly.
  • Variable names should start with letter or underscore.
  • Python is case-sensitive: name ≠ Name.
# Storing different types of data
x = 10          
name = "Alice"  
pi = 3.14       
is_active = True  

print(x, name, pi, is_active)

Python Data Types

  • int: Whole numbers
  • float: Decimal numbers
  • str: Text
  • bool: True/False values
age = 25           
height = 5.9       
city = "London"    
is_student = False 

print(age, height, city, is_student)

Check Data Type

x = 42
y = "Python"

print(type(x))  # <class 'int'>
print(type(y))  # <class 'str'>

Basic Operations

Arithmetic

a = 10
b = 3

print(a + b)  # 13
print(a - b)  # 7
print(a * b)  # 30
print(a / b)  # 3.333
print(a % b)  # 1

Comparison

print(a == b)  # False
print(a > b) # True
print(a < b) # False
