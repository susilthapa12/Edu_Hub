Python Lists

A list is an ordered collection of items. Lists are mutable (can be changed).

# Creating a list
fruits = ["apple", "banana", "cherry"]
print(fruits)

# Accessing elements
print(fruits[0])   # apple
print(fruits[-1])  # cherry

# Modifying elements
fruits[1] = "orange"
print(fruits)

# Adding elements
fruits.append("mango")
print(fruits)

# Removing elements
fruits.remove("apple")
print(fruits)

Python Tuples

A tuple is an ordered collection, but it is immutable.

# Creating a tuple
colors = ("red", "green", "blue")
print(colors)

# Accessing elements
print(colors[0])   # red
print(colors[-1])  # blue

# colors[0] = "yellow"  # This will give an error

Python Sets

A set is an unordered collection of unique elements. Sets are mutable.

# Creating a set
numbers = {1, 2, 3, 4, 4, 2}  # duplicates removed
print(numbers)

# Adding elements
numbers.add(5)
print(numbers)

# Removing elements
numbers.remove(3)
print(numbers)

# Set operations
evens = {2,4,6}
odds = {1,3,5}
print(evens.union(odds))        # {1,2,3,4,5,6}
print(evens.intersection(odds)) # {} (no common elements)

Python Dictionaries

A dictionary stores data as key-value pairs. Dictionaries are unordered but mutable.

# Creating a dictionary
person = {"name": "Alice", "age": 25, "city": "London"}
print(person)

# Accessing values
print(person["name"])  # Alice

# Adding or modifying values
person["age"] = 26
person["job"] = "Engineer"
print(person)

# Removing a key-value pair
del person["city"]
print(person)

# Looping through dictionary
for key, value in person.items():
    print(key, ":", value)
