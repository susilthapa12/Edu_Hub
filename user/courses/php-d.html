PHP Operators

Operators are used to perform operations on variables and values.

1. Arithmetic Operators

<?php
$a = 10;
$b = 3;

echo $a + $b; // Addition: 13
echo $a - $b; // Subtraction: 7
echo $a * $b; // Multiplication: 30
echo $a / $b; // Division: 3.3333
echo $a % $b; // Modulus: 1
echo $a ** $b; // Exponent: 1000
?>

2. Assignment Operators

<?php
$x = 5;
$x += 3; // $x = $x + 3 → 8
$x -= 2; // $x = $x - 2 → 6
$x *= 2; // $x = $x * 2 → 12
$x /= 4; // $x = $x / 4 → 3
$x %= 2; // $x = $x % 2 → 1
?>

3. Comparison Operators

<?php
$a = 5;
$b = 10;

var_dump($a == $b); // equal → false
var_dump($a != $b); // not equal → true
var_dump($a > $b);  // greater than → false
var_dump($a < $b);  // less than → true
var_dump($a >= $b); // greater or equal → false
var_dump($a <= $b); // less or equal → true
?>

4. Logical Operators

<?php
$x = true;
$y = false;

var_dump($x && $y); // AND → false
var_dump($x || $y); // OR → true
var_dump(!$x);      // NOT → false
?>

5. String Operators

<?php
$str1 = "Hello ";
$str2 = "World!";

echo $str1 . $str2;  // Concatenation: Hello World!
$str1 .= $str2;      // Append: $str1 = "Hello World!"
echo $str1;
?>

6. Increment / Decrement Operators

<?php
$a = 5;
echo ++$a; // Pre-increment: 6
echo $a++; // Post-increment: 6 (then $a becomes 7)
echo --$a; // Pre-decrement: 6
echo $a--; // Post-decrement: 6 (then $a becomes 5)
?>

7. Spaceship Operator (PHP 7+)

Used for combined comparison:

<?php
echo 1 <=> 1; // 0 (equal)
echo 1 <=> 2; // -1 (left smaller)
echo 2 <=> 1; // 1 (left bigger)
?>
