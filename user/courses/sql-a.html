SQL Basics

SQL (Structured Query Language) is used to manage and manipulate relational databases.

Before continuing, you should understand the concepts of tables, rows, and columns in databases.

SELECT Statement

The SELECT statement is used to retrieve data from a database table.

SELECT CustomerName, City, Country
FROM Customers;

This query retrieves the CustomerName, City, and Country columns from the Customers table.

WHERE Clause

The WHERE clause filters records based on conditions.

SELECT * FROM Customers
WHERE Country='Germany';

This query returns only customers from Germany.

ORDER BY Clause

Use ORDER BY to sort query results.

SELECT * FROM Customers
ORDER BY CustomerName ASC;

This query retrieves all customers sorted alphabetically by CustomerName.
