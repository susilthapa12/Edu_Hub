JavaScript Syntax

JavaScript syntax is the set of rules that define how programs are constructed. JavaScript programs are made up of statements, expressions, and keywords.

JavaScript Statements

JavaScript statements are instructions that are executed by the browser.

<!DOCTYPE html>
<html>
<body>
  <h2>JavaScript Statements</h2>
  <p id="demo"></p>

  <script>
    let x, y, z;    // Declare 3 variables
    x = 5;          // Assign the value 5 to x
    y = 6;          // Assign the value 6 to y
    z = x + y;      // Assign the sum of x and y to z
    document.getElementById("demo").innerHTML =
      "The value of z is " + z + ".";
  </script>
</body>
</html>

Semicolons

Semicolons separate JavaScript statements. They are optional if the statement is written on a new line, but best practice is to always use them.

let a = 10;
let b = 20;
let c = a + b;

Case Sensitivity

JavaScript is case-sensitive. Variables named lastname and lastName are two different variables.

let lastname = "Smith";
let lastName = "Johnson";

document.write(lastname);
document.write("<br>");
document.write(lastName);

JavaScript Comments

Comments can be used to explain code and make it more readable. JavaScript ignores comments.

// This is a single-line comment

/* 
   This is a 
   multi-line comment
*/
