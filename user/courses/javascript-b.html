JavaScript Variables

Variables are containers for storing data values. In JavaScript, you can declare variables using var, let, or const.

Declaring Variables

You can declare a variable using the let keyword:

<!DOCTYPE html>
<html>
<body>
  <p id="demo"></p>
  <script>
    let name = "Alice";
    document.getElementById("demo").innerHTML = name;
  </script>
</body>
</html>

var, let, and const

  • var - old way of declaring variables (not recommended).
  • let - modern way, can be reassigned.
  • const - used for constants, cannot be changed.
var city = "Delhi";
let age = 25;
const country = "India";

Reassigning Values

Variables declared with let can be changed:

<!DOCTYPE html>
<html>
<body>
  <p id="demo"></p>
  <script>
    let x = 10;
    x = 20; // reassign
    document.getElementById("demo").innerHTML = x;
  </script>
</body>
</html>

Constants

Variables declared with const cannot be reassigned:

const pi = 3.1416;
// pi = 3.14; ❌ This will cause an error
