JavaScript Variables

Variables are containers for storing data values. In JavaScript you can declare variables using var, let, or const. Below you'll find live examples, copy buttons, and Try-It-Yourself frames.

Declaring Variables

Example using let to declare a variable and display it.

Example
<!DOCTYPE html>
<html>
<body>
  <p id="demo"></p>
  <script>
    let name = "Alice";
    document.getElementById("demo").innerHTML = name;
  </script>
</body>
</html>

var, let, and const

Quick differences and recommended usage.

var — legacy let — block-scoped const — immutable binding 
var city = "Delhi";
let age = 25;
const country = "India";

Reassigning Values

Variables declared with let can be reassigned.

<!DOCTYPE html>
<html>
<body>
  <p id="demo"></p>
  <script>
    let x = 10;
    x = 20; // reassign
    document.getElementById("demo").innerHTML = x;
  </script>
</body>
</html>

Constants

Variables declared with const cannot be reassigned.

const pi = 3.1416;
// pi = 3.14; ❌ This will cause an error
