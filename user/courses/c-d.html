C Programming Operators

Operators are symbols that perform operations on variables and values.

1. Arithmetic Operators

Used for mathematical operations:

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus (remainder)
#include <stdio.h>

int main() {
    int a = 10, b = 3;
    printf("a + b = %d\n", a + b);
    printf("a - b = %d\n", a - b);
    printf("a * b = %d\n", a * b);
    printf("a / b = %d\n", a / b);
    printf("a % b = %d\n", a % b);
    return 0;
}

2. Assignment Operators

  • = Assign value
  • += Add and assign
  • -= Subtract and assign
  • *= Multiply and assign
  • /= Divide and assign
  • %= Modulus and assign
int x = 5;
x += 3; // x = 8
x *= 2; // x = 16

3. Relational Operators

  • == Equal to
  • != Not equal to
  • < Less than
  • <= Less than or equal to
  • > Greater than
  • >= Greater than or equal to
int a = 5, b = 3;
printf("%d\n", a == b); // 0 (false)
printf("%d\n", a != b); // 1 (true)

4. Logical Operators

  • && AND
  • || OR
  • ! NOT
int a = 5, b = 3;
if(a > 0 && b > 0) {
    printf("Both positive\n");
}
if(a < 0 || b < 0) {
    printf("At least one negative\n");
} else {
    printf("None negative\n");
}

5. Increment & Decrement Operators

  • ++ Increment by 1
  • -- Decrement by 1
int a = 5;
a++; // a = 6
a--; // a = 5

6. Conditional (Ternary) Operator

int a = 5, b = 3;
int max = (a > b) ? a : b; // max = 5
