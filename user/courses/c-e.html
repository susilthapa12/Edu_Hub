C Programming Loops

Loops allow executing a block of code multiple times. C supports for, while, and do-while loops.

1. For Loop

#include <stdio.h>

int main() {
    int i;
    for(i = 1; i <= 5; i++) {
        printf("%d\n", i);
    }
    return 0;
}

Explanation: Initializes i = 1, checks i <= 5, executes the loop, then increments i++.

2. While Loop

#include <stdio.h>

int main() {
    int i = 1;
    while(i <= 5) {
        printf("%d\n", i);
        i++;
    }
    return 0;
}

Explanation: Executes as long as the condition i <= 5 is true.

3. Do-While Loop

#include <stdio.h>

int main() {
    int i = 1;
    do {
        printf("%d\n", i);
        i++;
    } while(i <= 5);
    return 0;
}

Explanation: Executes the loop first, then checks the condition. Ensures the loop runs at least once.

4. Nested Loops

#include <stdio.h>

int main() {
    int i, j;
    for(i = 1; i <= 3; i++) {
        for(j = 1; j <= 3; j++) {
            printf("%d ", j);
        }
        printf("\n");
    }
    return 0;
}

Output:

1 2 3
1 2 3
1 2 3
