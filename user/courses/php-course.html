PHP Introduction

PHP code is executed on the server, and the result is returned to the browser as plain HTML.

What You Should Already Know

Before learning PHP, you should have basic knowledge of:

  • HTML
  • CSS
  • JavaScript

If you need to learn these first, check the tutorials on our Home page.

What is PHP?

  • PHP stands for "PHP: Hypertext Preprocessor"
  • It is an open-source scripting language
  • PHP code is executed on the server
  • Free to download and use

PHP is powerful and popular: it runs WordPress, powers large social networks, and is simple enough for beginners!

What is a PHP File?

  • Can contain text, HTML, CSS, JavaScript, and PHP code
  • PHP code is executed on the server and returned as plain HTML
  • File extension is .php

What Can PHP Do?

  • Generate dynamic page content
  • Create, open, read, write, delete, and close files on the server
  • Collect form data and send/receive cookies
  • Add, delete, modify data in your database
  • Control user access and encrypt data
  • Output not only HTML but also images, PDF, XML, or any text

Why PHP?

  • Runs on Windows, Linux, Unix, Mac OS X, etc.
  • Compatible with most servers (Apache, IIS, etc.)
  • Supports a wide range of databases
  • Free and efficient to run
  • Easy to learn

What's New in PHP 7?

  • Much faster than PHP 5.6
  • Improved error handling
  • Stricter type declarations for function arguments
  • New operators like the spaceship operator (<=>)

Basic PHP Example

PHP code is written inside <?php ... ?> tags.

<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>
