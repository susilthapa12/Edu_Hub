Importing Data

Pandas can read data from CSV, Excel, JSON, SQL databases, and more.

import pandas as pd

# CSV
df_csv = pd.read_csv('data.csv')

# Excel
df_excel = pd.read_excel('data.xlsx')

# JSON
df_json = pd.read_json('data.json')

# Display first 5 rows
print(df_csv.head())

Filtering & Sorting

# Filter rows
filtered = df_csv[df_csv['Age'] > 25]

# Sort by column
sorted_df = df_csv.sort_values(by='Salary', ascending=False)

print(filtered)
print(sorted_df)

Grouping & Aggregation

# Group by department and calculate mean salary
grouped = df_csv.groupby('Department')['Salary'].mean()
print(grouped)

# Multiple aggregations
agg = df_csv.groupby('Department').agg({'Salary': ['mean','max'], 'Age':'min'})
print(agg)

Data Cleaning

# Fill missing values
df_csv['Salary'].fillna(df_csv['Salary'].mean(), inplace=True)

# Drop rows with missing values
df_csv.dropna(inplace=True)

# Remove duplicates
df_csv.drop_duplicates(inplace=True)

Basic Plotting

Pandas integrates with Matplotlib for quick plotting:

import matplotlib.pyplot as plt

# Plot Age histogram
df_csv['Age'].hist()
plt.show()

# Plot Salary by Department
df_csv.groupby('Department')['Salary'].mean().plot(kind='bar')
plt.show()
