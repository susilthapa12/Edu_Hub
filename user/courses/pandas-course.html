Pandas Introduction

Pandas is a Python library used for working with data sets. It was created by Wes McKinney in 2008.

The name "Pandas" comes from "Panel Data" and "Python Data Analysis".

Why Use Pandas?

  • Analyze large datasets and find patterns
  • Clean messy data
  • Extract relevant data for data science
  • Compute statistical values like mean, max, min, correlations

Pandas Basic Objects

Pandas has two main data structures: Series and DataFrame.

import pandas as pd

# Series: One-dimensional
s = pd.Series([10, 20, 30])
print(s)

# DataFrame: Two-dimensional
data = {'Name': ['Alice', 'Bob'], 'Age': [25, 30]}
df = pd.DataFrame(data)
print(df)

Data Cleaning

Pandas can handle missing or invalid data efficiently.

import pandas as pd
import numpy as np

data = {'Name': ['Alice', 'Bob', None], 'Age': [25, None, 30]}
df = pd.DataFrame(data)

# Drop rows with missing values
df_clean = df.dropna()
print(df_clean)

# Fill missing values
df_filled = df.fillna({'Name': 'Unknown', 'Age': 0})
print(df_filled)

Basic Operations

import pandas as pd

data = {'A': [10, 20], 'B': [30, 40]}
df = pd.DataFrame(data)

# Select a column
print(df['A'])

# Add a new column
df['C'] = df['A'] + df['B']
print(df)

# Filter rows
print(df[df['C'] > 40])
