HTML Styles

The style attribute allows you to add CSS to HTML elements: colors, fonts, sizes, and more.

Red and Blue Text

<p style="color:red;">I am Red</p>
<p style="color:blue;">I am Blue</p>

Big Text

<p style="font-size:200%;">I am Big</p>

Background Colors

<body style="background-color:powderblue;">
  <h1>Heading</h1>
  <p>Paragraph</p>
</body>

<h1 style="background-color:powderblue;">Heading</h1>
<p style="background-color:tomato;">Paragraph</p>

Fonts & Text Sizes

<h1 style="font-family:verdana; font-size:300%;">Heading</h1>
<p style="font-family:courier; font-size:160%;">Paragraph</p>

Text Alignment

<h1 style="text-align:center;">Centered Heading</h1>
<p style="text-align:center;">Centered paragraph</p>

Chapter Summary

  • Use style for inline element styling
  • Use color for text colors
  • Use background-color for backgrounds
  • Use font-family for fonts
  • Use font-size for text sizes
  • Use text-align for alignment
