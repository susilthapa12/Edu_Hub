AngularJS Expressions

AngularJS expressions are written inside double braces {{ expression }} and automatically bind data to the view:

<div ng-app="">
  <p>5 + 5 = {{ 5 + 5 }}</p>
</div>
<div ng-app="">
  <p>Name: <input type="text" ng-model="name"></p>
  <p>Hello {{name}}!</p>
</div>

AngularJS Modules

Modules define the structure of an AngularJS application:

var app = angular.module('myApp', []);

The first parameter is the module name, the second is a list of dependencies (empty array if none).

AngularJS Controllers

Controllers manage application data and logic. They are attached to modules:

app.controller('myCtrl', function($scope) {
  $scope.firstName = "John";
  $scope.lastName = "Doe";
});

HTML usage:

<div ng-app="myApp" ng-controller="myCtrl">
  First Name: <input type="text" ng-model="firstName"><br>
  Last Name: <input type="text" ng-model="lastName"><br>
  Full Name: {{firstName + " " + lastName}}
</div>

Summary

  • Expressions {{ }} bind AngularJS variables to the view.
  • Modules define the AngularJS application structure.
  • Controllers manage data and connect it to the HTML.
  • ng-model binds form inputs to controller data.
