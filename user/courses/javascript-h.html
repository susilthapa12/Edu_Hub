JavaScript Arrays

An array is a special variable that holds multiple values. Arrays are ordered and zero-indexed.

Creating Arrays

use [] 
<!DOCTYPE html>
<html>
<body>
  <p id="demo"></p>
  <script>
    let fruits = ["Apple", "Banana", "Orange"];
    document.getElementById("demo").innerText = fruits;
  </script>
</body>
</html>

Accessing Array Items

Items are accessed by index (starting at 0).

let fruits = ["Apple", "Banana", "Orange"];
document.write(fruits[0]); // Apple
document.write("<br>" + fruits[1]); // Banana

Changing Array Items

let fruits = ["Apple", "Banana", "Orange"];
fruits[1] = "Mango";
document.write(fruits); // Apple,Mango,Orange

Array Length

let fruits = ["Apple", "Banana", "Orange"];
document.write("Total fruits: " + fruits.length);

Common Array Methods

let fruits = ["Apple", "Banana"];
fruits.push("Orange"); // add at end
document.write(fruits + "<br>");

fruits.pop(); // remove last
document.write(fruits + "<br>");

fruits.unshift("Mango"); // add at start
document.write(fruits + "<br>");

fruits.shift(); // remove first
document.write(fruits);

Looping Through Arrays

let fruits = ["Apple", "Banana", "Orange"];
for (let i = 0; i < fruits.length; i++) {
  document.write(fruits[i] + "<br>");
}
