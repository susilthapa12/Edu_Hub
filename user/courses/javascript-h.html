JavaScript Arrays

An array is a special variable that can hold more than one value at a time. For example, you can store a list of names, numbers, or even mixed data.

Creating Arrays

Arrays are written with square brackets [].

let fruits = ["Apple", "Banana", "Orange"];
document.write(fruits);

Accessing Array Items

You can access items in an array using their index (starting at 0). 

let fruits = ["Apple", "Banana", "Orange"];
document.write(fruits[0]); // Apple
document.write("<br>" + fruits[1]); // Banana

Changing Array Items

You can change an item by using its index.

let fruits = ["Apple", "Banana", "Orange"];
fruits[1] = "Mango";
document.write(fruits); // Apple,Mango,Orange

Array Length

The length property tells you how many items are in an array. 

let fruits = ["Apple", "Banana", "Orange"];
document.write("Total fruits: " + fruits.length);

Common Array Methods

Here are some useful methods:

  • push() – adds an item to the end
  • pop() – removes the last item
  • shift() – removes the first item
  • unshift() – adds an item to the beginning
let fruits = ["Apple", "Banana"];
fruits.push("Orange"); // add at end
document.write(fruits + "<br>");

fruits.pop(); // remove last
document.write(fruits + "<br>");

fruits.unshift("Mango"); // add at start
document.write(fruits + "<br>");

fruits.shift(); // remove first
document.write(fruits);

Looping Through Arrays

You can use a for loop to go through all the items in an array. 

let fruits = ["Apple", "Banana", "Orange"];
for (let i = 0; i < fruits.length; i++) {
  document.write(fruits[i] + "<br>");
}
