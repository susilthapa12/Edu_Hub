AngularJS Introduction

AngularJS is a JavaScript framework that extends HTML with Directives and binds data to HTML with Expressions.

<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>

Basic Example

Hello {{name}}!

<div ng-app="" ng-init="name='John'">
  <input type="text" ng-model="name">
  <p>Hello {{name}}!</p>
</div>

Directives

Directives are HTML attributes with an ng- prefix.

<div ng-app="" ng-init="firstName='John'">
  <p>The name is <span ng-bind="firstName"></span></p>
</div>

Expressions

Expressions are written inside double braces: {{ expression }}

<div ng-app="">
  <p>My first expression: {{ 5 + 5 }}</p>
</div>

Applications & Controllers

Full Name: {{app.firstName + ' ' + app.lastName}}

<script>
angular.module('demoApp', []).controller('AppCtrl', function() {
  this.firstName = "John";
  this.lastName = "Doe";
});
</script>
