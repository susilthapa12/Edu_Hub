JavaScript Operators

Operators are symbols used to perform operations on values and variables. Let’s go through the most common types of operators in JavaScript.

Arithmetic Operators

Used for basic math:

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus
  • ** Exponentiation
Copied!
<!DOCTYPE html>
<html>
<body>
  <p id="demo"></p>
  <script>
    let x = 10;
    let y = 3;
    let result = x + y; // 13
    document.getElementById("demo").innerHTML = "10 + 3 = " + result;
  </script>
</body>
</html>

Assignment Operators

Copied!
let x = 5;
x += 2; // x = x + 2
x *= 3; // x = x * 3
document.write(x); // 21

Comparison Operators

  • == Equal to (ignores type)
  • === Strict equal (checks type)
  • != Not equal
  • !== Strict not equal
  • > Greater than
  • < Less than
  • >= Greater or equal
  • <= Less or equal
Copied!
let a = 5;
let b = "5";
document.write(a == b);   // true
document.write("<br>");
document.write(a === b);  // false

Logical Operators

Copied!
let age = 20;
document.write(age >= 18 && age <= 30); // true

String Operators

Copied!
let firstName = "John";
let lastName = "Doe";
document.write(firstName + " " + lastName);
