JavaScript Operators

Operators are symbols used to perform operations on values and variables. Let’s go through the most common types of operators in JavaScript.

Arithmetic Operators

Used for basic math:

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus (remainder)
  • ** Exponentiation (power)
<!DOCTYPE html>
<html>
<body>
  <p id="demo"></p>
  <script>
    let x = 10;
    let y = 3;
    let result = x + y; // 13
    document.getElementById("demo").innerHTML = "10 + 3 = " + result;
  </script>
</body>
</html>

Assignment Operators

Used to assign values to variables:

let x = 5;
x += 2; // same as x = x + 2
x *= 3; // same as x = x * 3
document.write(x); // 21

Comparison Operators

Used to compare values. Returns true or false:

  • == Equal to (ignores type)
  • === Strict equal (checks value and type)
  • != Not equal
  • !== Strict not equal
  • > Greater than
  • < Less than
  • >= Greater than or equal
  • <= Less than or equal
let a = 5;
let b = "5";
document.write(a == b);   // true
document.write("<br>");
document.write(a === b);  // false

Logical Operators

Used to combine conditions:

  • && AND (true if both are true)
  • || OR (true if at least one is true)
  • ! NOT (reverses the result)
let age = 20;
document.write(age >= 18 && age <= 30); // true

String Operators

The + operator can also join (concatenate) strings:

let firstName = "John";
let lastName = "Doe";
document.write(firstName + " " + lastName);
