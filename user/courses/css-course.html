CSS Introduction

CSS is the language we use to style a Web page.

What is CSS? #

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are displayed on screen, paper, or other media
  • CSS saves work by controlling layout of multiple pages at once
  • External stylesheets are stored in CSS files

CSS Demo #

One HTML page can use multiple stylesheets. Click the links to switch styles.

Why Use CSS? #

CSS defines styles for web pages including design, layout, and device-specific variations.

Example Code # 

body {
  background-color: lightblue;
}

h1 {
  color: white;
  text-align: center;
}

p {
  font-family: verdana;
  font-size: 20px;
}
