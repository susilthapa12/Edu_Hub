Time Series in Pandas

Pandas provides powerful tools to work with date and time data.

import pandas as pd

# Create date range
dates = pd.date_range('2025-01-01', periods=6, freq='D')
data = pd.DataFrame({'Sales':[250,300,400,500,450,600]}, index=dates)

# Accessing by date
print(data['2025-01-03'])

# Resampling by week
weekly_sales = data.resample('W').sum()
print(weekly_sales)

Multi-Indexing

Multi-indexing allows hierarchical indexing for complex datasets.

arrays = [['Store1','Store1','Store2','Store2'], ['2025-01','2025-02','2025-01','2025-02']]
index = pd.MultiIndex.from_arrays(arrays, names=('Store','Month'))

data = pd.DataFrame({'Sales':[200,250,300,350]}, index=index)
print(data)

# Access data for Store1
print(data.loc['Store1'])

Exporting Data

Pandas allows exporting data to CSV, Excel, and JSON formats.

# Export to CSV
data.to_csv('sales.csv')

# Export to Excel
data.to_excel('sales.xlsx')

# Export to JSON
data.to_json('sales.json')

Grouping & Aggregations

# Sample data
df = pd.DataFrame({
    'Store':['S1','S1','S2','S2'],
    'Month':['Jan','Feb','Jan','Feb'],
    'Sales':[200,250,300,350]
})

# Group by Store and sum
grouped = df.groupby('Store')['Sales'].sum()
print(grouped)
