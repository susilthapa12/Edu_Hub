JavaScript Data Types

JavaScript variables can hold different types of data. The main types are: String, Number, Boolean, Array, Object, Undefined, and Null.

  • String â€“ text like "Hello"
  • Number â€“ numbers like 42 or 3.14
  • Boolean â€“ true or false
  • Array â€“ a list of values
  • Object â€“ a collection of key/value pairs
  • Undefined â€“ a variable with no value
  • Null â€“ empty value

Numbers

Numbers can be integers or decimals:

let x = 10;
let y = 3.14;
document.write(x + y);

Booleans

Booleans can only be true or false:

let isOnline = true;
let isLoggedIn = false;
document.write(isOnline);

Arrays

Arrays are used to store multiple values in one variable:

<!DOCTYPE html>
<html>
<body>
  <p id="demo"></p>
  <script>
    let colors = ["Red", "Green", "Blue"];
    document.getElementById("demo").innerHTML = colors[0]; // Red
  </script>
</body>
</html>

Objects

Objects store data in key/value pairs:

<!DOCTYPE html>
<html>
<body>
  <p id="demo"></p>
  <script>
    let person = {firstName:"John", lastName:"Doe", age:30};
    document.getElementById("demo").innerHTML = person.firstName + " " + person.lastName;
  </script>
</body>
</html>

Undefined & Null

If a variable is declared but has no value, it is undefined. null means "no value".

let a;
let b = null;
document.write(a);   // undefined
document.write("<br>");
document.write(b);   // null
