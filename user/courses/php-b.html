PHP Syntax Basics

PHP code is executed on the server and enclosed within special tags: <?php ... ?>

PHP Tags

PHP can be embedded in HTML using:

  • Standard tags: <?php ... ?> (most common)
  • Short echo tags: <?= ... ?> (used for quick echo)
<?php
echo "Hello World!";
?>

<?= "Hello World!" ?>

PHP Comments

Comments are notes in code and are ignored by PHP.

  • Single-line comments using // or #
  • Multi-line comments using /* ... */
<?php
// This is a single-line comment
# This is also a comment

/*
This is a multi-line comment
It can span multiple lines
*/
echo "Hello Comments!";
?>

PHP Statements

PHP statements are terminated with a semicolon ;.

<?php
echo "Statement 1";
echo "Statement 2";
?>

Case Sensitivity

Function names in PHP are case-insensitive, but variables are case-sensitive.

<?php
$color = "red";
$Color = "blue"; // different variable

echo $color; // outputs: red
echo $Color; // outputs: blue
?>
