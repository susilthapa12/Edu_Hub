JavaScript Functions

A function is a block of code that performs a specific task. You can call a function whenever you need it, instead of repeating code.

Defining a Function

A function is defined with the function keyword, followed by a name and parentheses (). 

<!DOCTYPE html>
<html>
<body>
  <script>
    function sayHello() {
      alert("Hello, World!");
    }
    sayHello(); // Calling the function
  </script>
</body>
</html>

Function Parameters

Functions can take parameters (inputs) and use them inside the code. 

function greet(name) {
  document.write("Hello, " + name + "!<br>");
}
greet("Alice");
greet("Bob");

Returning Values

Functions can return values using the return keyword. 

function add(a, b) {
  return a + b;
}
let result = add(5, 3);
document.write("The sum is: " + result);

Anonymous Functions

Functions can also be stored in variables. These are called anonymous functions. 

let multiply = function(x, y) {
  return x * y;
};
document.write("Result: " + multiply(4, 5));

Arrow Functions

Arrow functions are a shorter way to write functions. 

let square = (n) => n * n;
document.write("Square of 6: " + square(6));
