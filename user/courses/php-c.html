PHP Variables

Variables in PHP are used to store data. A variable starts with a dollar sign $.

<?php
$name = "Alice"; // string
$age = 25;       // integer
$height = 5.7;   // float
$isStudent = true; // boolean

echo $name;
echo $age;
echo $height;
echo $isStudent;
?>

Variable names are case-sensitive: $name and $Name are different variables.

PHP Data Types

Common data types in PHP include:

  • String: sequence of characters, e.g., "Hello"
  • Integer: whole numbers, e.g., 42
  • Float / Double: decimal numbers, e.g., 3.14
  • Boolean: true or false
  • Array: collection of values
  • Object: instance of a class
  • NULL: variable with no value

Example: Data Types

<?php
$firstName = "Bob";       // string
$age = 30;                // integer
$gpa = 3.8;               // float
$isEnrolled = false;      // boolean
$fruits = array("apple", "banana", "cherry"); // array
$person = null;           // null

echo $firstName;
echo $age;
echo $gpa;
echo $isEnrolled;
print_r($fruits); // prints array
var_dump($person); // prints NULL
?>

Variable Rules

  • Start with $
  • Followed by a letter or underscore
  • Can contain letters, numbers, and underscores
  • Cannot start with a number
  • Case-sensitive
