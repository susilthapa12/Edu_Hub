Advanced CSS: Box Model & Layout

The CSS Box Model

Every HTML element is a rectangular box. The CSS box model consists of: content, padding, border, and margin. 

div {
  width: 300px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
}

Padding

Padding is the space between the content and the border. You can define different paddings for each side: 

div {
  padding-top: 10px;
  padding-right: 15px;
  padding-bottom: 10px;
  padding-left: 15px;
}

Margin

Margin is the space outside the border, between the element and other elements: 

div {
  margin: 20px;
}

Borders

Borders surround the padding and content. You can style borders with color, width, and style: 

div {
  border-width: 3px;
  border-style: solid;
  border-color: red;
}

Display Property

The display property defines how an element is displayed: 

div {
  display: block;   /* Block element */
}

span {
  display: inline;  /* Inline element */
}

ul {
  display: flex;    /* Flex container */
}

Positioning

CSS positioning controls how elements are positioned on the page: 

div {
  position: relative; /* relative to normal position */
}

div.fixed {
  position: fixed;    /* fixed relative to viewport */
}

div.absolute {
  position: absolute; /* relative to nearest positioned ancestor */
}

Chapter Summary

  • CSS Box Model: content, padding, border, margin
  • Padding defines inner spacing
  • Margin defines outer spacing
  • Borders surround content and padding
  • Display property controls element type and layout
  • Position property allows relative, absolute, and fixed placement
