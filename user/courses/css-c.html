Advanced CSS: Box Model & Layout

Learn how content, padding, border, and margin interact and control layouts.

CSS Box Model

div {
  width: 300px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
}

Padding

div {
  padding-top: 10px;
  padding-right: 15px;
  padding-bottom: 10px;
  padding-left: 15px;
}

Margin

div {
  margin: 20px;
}

Borders

div {
  border-width: 3px;
  border-style: solid;
  border-color: red;
}

Display Property

div { display: block; }
span { display: inline; }
ul { display: flex; }

Positioning

div { position: relative; }
div.fixed { position: fixed; }
div.absolute { position: absolute; }

Chapter Summary

  • CSS Box Model: content, padding, border, margin
  • Padding defines inner spacing
  • Margin defines outer spacing
  • Borders surround content and padding
  • Display property controls element type and layout
  • Position property allows relative, absolute, and fixed placement
