JavaScript Objects

Objects hold multiple related values as properties (key/value pairs). Properties can be primitive values, arrays, or even functions (methods).

Creating an Object

object literal 
<!DOCTYPE html>
<html>
<body>
  <p id="out"></p>
  <script>
    let person = {
      name: "John",
      age: 25,
      city: "New York"
    };
    document.getElementById("out").innerText = person.name + ", " + person.age + ", " + person.city;
  </script>
</body>
</html>

Accessing Object Properties

Use dot obj.key or bracket obj['key'] notation.

let person = {name: "Alice", age: 30};
document.write(person.name); // Alice
document.write("<br>" + person["age"]); // 30

Changing Object Properties

let person = {name: "Alice", age: 30};
person.age = 31;
person["name"] = "Bob";
document.write(person.name + ", " + person.age); // Bob, 31

Adding Properties

let person = {name: "Alice"};
person.city = "Paris";
document.write(person.name + ", " + person.city); // Alice, Paris

Deleting Properties

let person = {name: "Alice", age: 30};
delete person.age;
document.write(person.age); // undefined

Object Methods

Objects may contain functions called methods; inside methods `this` refers to the object.

let person = {
  name: "John",
  greet: function() {
    return "Hello, " + this.name;
  }
};
document.write(person.greet()); // Hello, John

Looping Through Objects

Use for...in to iterate keys (own/prototype keys differ; prefer `Object.keys()` for own keys).

let person = {name: "Alice", age: 30, city: "Paris"};
for (let key in person) {
  document.write(key + ": " + person[key] + "<br>");
}
