JavaScript Objects

Objects are variables that can hold **multiple related values** as properties. Each property has a key (name) and a value. Objects can also contain **functions** called methods.

Creating an Object

Use curly braces {} to create an object.

let person = {
  name: "John",
  age: 25,
  city: "New York"
};
document.write(person.name + ", " + person.age + ", " + person.city);

Accessing Object Properties

You can access values using dot . or bracket [] notation.

let person = {name: "Alice", age: 30};
document.write(person.name); // Alice
document.write(person["age"]); // 30

Changing Object Properties

let person = {name: "Alice", age: 30};
person.age = 31;
person["name"] = "Bob";
document.write(person.name + ", " + person.age); // Bob, 31

Adding Properties

let person = {name: "Alice"};
person.city = "Paris";
document.write(person.name + ", " + person.city); // Alice, Paris

Deleting Properties

let person = {name: "Alice", age: 30};
delete person.age;
document.write(person.age); // undefined

Object Methods

Objects can have functions as values. These are called methods.

let person = {
  name: "John",
  greet: function() {
    return "Hello, " + this.name;
  }
};
document.write(person.greet()); // Hello, John

Looping Through Objects

Use for...in to loop through object properties.

let person = {name: "Alice", age: 30, city: "Paris"};
for (let key in person) {
  document.write(key + ": " + person[key] + "<br>");
}
