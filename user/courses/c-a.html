C Syntax Basics

Here is how a basic C program is structured:

1. Basic Program Structure

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

Explanation:

  • #include <stdio.h> - includes the standard input/output library.
  • int main() - the main function where the program starts.
  • printf() - prints output to the screen.
  • return 0; - ends the program.

2. Comments in C

Comments are used to explain code and are ignored by the compiler.

// This is a single-line comment

/*
   This is a
   multi-line comment
*/

3. C Statements

Each line of code is called a statement and ends with a semicolon ;.

int a = 5;
int b = 10;
int sum = a + b;
printf("Sum = %d", sum);

4. Case Sensitivity

C is case-sensitive: Var and var are different variables.
