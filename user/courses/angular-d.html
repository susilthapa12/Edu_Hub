AngularJS Filters

Filters format the value of an expression for display to the user. They can be used in expressions and directives.

<div ng-app="">
  <p>Original: {{ name }}</p>
  <p>Uppercase: {{ name | uppercase }}</p>
  <p>Lowercase: {{ name | lowercase }}</p>
  <p>Currency: {{ amount | currency }}</p>
</div>

AngularJS provides built-in filters like uppercase, lowercase, currency, date, and more.

AngularJS Services

Services are reusable objects that provide functionality across the app, like $http for AJAX calls.

var app = angular.module('myApp', []);
app.controller('myCtrl', function($scope, $http) {
  $http.get('data.json')
    .then(function(response) {
      $scope.users = response.data;
    });
});

Services are injected into controllers or other components to handle logic or data access.

Summary

  • Filters format data in AngularJS expressions.
  • Common filters: uppercase, lowercase, currency, date.
  • Services provide reusable logic or data access.
  • $http is a built-in service for making AJAX requests.
