Git Branching

Branches let you work on different versions of a project simultaneously.

# List branches
git branch

# Create a new branch
git branch feature-branch

# Switch to the branch
git checkout feature-branch

Merging Branches

Once work is done, merge branches back into the main branch:

# Switch to main branch
git checkout main

# Merge changes from feature-branch
git merge feature-branch

If conflicts appear, Git will prompt you to resolve them manually.

Working with Remote Repositories

Remote repositories allow you to collaborate online with services like GitHub, GitLab, or Bitbucket.

# Add a remote repository
git remote add origin https://github.com/username/repo.git

# Push local commits to remote
git push -u origin main

# Fetch latest changes from remote
git pull origin main
