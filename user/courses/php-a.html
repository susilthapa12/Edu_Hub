PHP Installation & Setup

1. Requirements

  • Web Server (Apache, Nginx, IIS, etc.)
  • PHP runtime (PHP 7.x or higher recommended)
  • Database (MySQL, MariaDB, PostgreSQL) if you plan to store data

2. Installation on Windows

The easiest way is to install a pre-packaged environment like XAMPP or WampServer:

  • Download XAMPP or WampServer
  • Run the installer and follow the instructions
  • Start Apache and MySQL from the control panel
  • Place PHP files in the htdocs folder (for XAMPP)

3. Installation on Linux (Ubuntu/Debian)

Use the terminal to install Apache and PHP:

sudo apt update
sudo apt install apache2
sudo apt install php libapache2-mod-php
sudo systemctl restart apache2

PHP files are usually placed in /var/www/html/.

4. Installation on macOS

macOS comes with PHP pre-installed. To check version:

php -v

For more control, you can install PHP via Homebrew:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install php

5. Verify PHP Installation

Create a file named info.php in your server directory with the following content:

<?php
phpinfo();
?>

Open http://localhost/info.php in your browser. You should see the PHP configuration page.

6. Editor Recommendation

  • Visual Studio Code (VS Code) – lightweight, free, and supports PHP extensions
  • PHPStorm – full-featured, professional IDE (paid)
