Installing Git

Git can be installed on Windows, macOS, and Linux. Official downloads are available at git-scm.com.

After installation, verify by running:

git --version

Initializing a Git Repository

To start tracking a project with Git, go to your project folder and run:

git init

This creates a hidden .git folder where Git stores all version history.

Staging and Committing Changes

After modifying files, you need to stage them before committing:

git add filename.txt
git add .

Commit the changes with a descriptive message:

git commit -m "Initial commit"

Check the commit history:

git log

Checking Repository Status

See which files are staged, unstaged, or untracked:

git status
