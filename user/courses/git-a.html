Installing Git

Git can be installed on Windows, macOS, and Linux. Official downloads: git-scm.com.

Verify installation:

git --version

Initializing a Git Repository

Go to your project folder and run:

git init

This creates a hidden .git folder for version history.

Staging and Committing Changes

Stage modified files before committing:

git add filename.txt
git add .

Commit with a message:

git commit -m "Initial commit"

Check commit history:

git log

Click to see Advanced Git Commands

git rebase
git cherry-pick
git stash
git revert

Checking Repository Status

See which files are staged, unstaged, or untracked:

git status
