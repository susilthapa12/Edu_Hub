Python for Loop

The for loop is used to iterate over a sequence (like a list, tuple, string, or range).

# Loop through a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# Loop with range
for i in range(5):
    print("Number:", i)

Python while Loop

The while loop runs as long as a condition is true.

# While loop example
count = 0
while count < 5:
    print("Count is:", count)
    count += 1

break and continue

The break statement stops the loop immediately.
The continue statement skips the current iteration and moves to the next. 

# Using break
for i in range(10):
    if i == 5:
        break
    print(i)

# Using continue
for i in range(10):
    if i % 2 == 0:
        continue
    print(i)

Nested Loops

A nested loop is a loop inside another loop.

# Nested for loops
for i in range(3):
    for j in range(2):
        print("i =", i, ", j =", j)
