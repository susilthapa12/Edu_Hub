Your First React App

We will create a simple React application using Create React App.

  1. Make sure you have Node.js and npm installed.
  2. Open your terminal and run:
npx create-react-app my-first-app
  1. Navigate into your project folder:
cd my-first-app
  1. Start the development server:
npm start

Understanding the App Structure

  • public/ – static files like index.html
  • src/ – source code for components and main app
  • package.json – project dependencies and scripts

Your First Component

Open src/App.js and replace its content with:

import React from 'react';

function App() {
  return (
    <div>
      <h1>Hello, React!</h1>
      <p>Welcome to your first React application.</p>
    </div>
  );
}

export default App;

Save the file and your browser will automatically reload showing your message.

Summary

  • Set up a new React project using Create React App.
  • Learned the folder structure of a React app.
  • Created your first React component and rendered it in the browser.
