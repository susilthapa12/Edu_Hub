Advanced CSS: Flexbox, Grid & Responsive Design

CSS Flexbox

Flexbox is a layout model that allows you to arrange elements in rows or columns with flexible sizing and alignment. 

.container {
  display: flex;
  flex-direction: row; /* row or column */
  justify-content: space-between;
  align-items: center;
}

.item {
  flex: 1; /* flexible size */
  margin: 5px;
}

CSS Grid

Grid is a two-dimensional layout system, allowing you to create rows and columns. 

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-gap: 10px;
}

.grid-item {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
}

Responsive Design

Responsive design makes web pages look good on all devices by using flexible layouts and media queries. 

@media screen and (max-width: 600px) {
  .container {
    flex-direction: column;
  }

  .grid-container {
    grid-template-columns: 1fr;
  }
}

Chapter Summary

  • Flexbox is for one-dimensional layouts (row or column)
  • Grid is for two-dimensional layouts (rows and columns)
  • Use flex and grid properties to control sizing and alignment
  • Media queries make layouts responsive on different devices
  • Combining Flexbox, Grid, and media queries allows advanced, adaptive designs
