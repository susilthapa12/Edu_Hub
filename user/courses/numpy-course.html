What is NumPy?

NumPy is a Python library for arrays, linear algebra, matrices, and fast mathematical computation.

Created by Travis Oliphant in 2005, NumPy is open-source.

Stands for Numerical Python.

Why Use NumPy?

NumPy arrays are faster than Python lists:

  • Arrays are contiguous in memory.
  • Optimized for CPU architectures.
  • Efficient operations on large datasets.
import numpy as np

arr = np.array([1, 2, 3, 4, 5])
print("Array:", arr)
print("Sum:", np.sum(arr))

Language Behind NumPy

NumPy is written in Python, but heavy computations are in C or C++.

Source code: GitHub

Example: NumPy Array Operations

import numpy as np

a = np.array([1, 2, 3])
b = np.array([4, 5, 6])

print("Addition:", a + b)
print("Dot Product:", np.dot(a, b))
