Advanced CSS: Animations, Transitions & Pseudo-Classes

CSS Transitions

Transitions allow changes to property values to occur smoothly over a specified duration. 

button {
  background-color: blue;
  color: white;
  transition: background-color 0.5s ease;
}

button:hover {
  background-color: green;
}

CSS Animations

Animations let you create more complex motion effects by defining keyframes. 

@keyframes slide {
  0% { transform: translateX(0); }
  50% { transform: translateX(100px); }
  100% { transform: translateX(0); }
}

.box {
  width: 100px;
  height: 100px;
  background-color: red;
  animation: slide 2s infinite;
}

Pseudo-Classes

Pseudo-classes define the special state of an element. For example, :hover, :focus, :nth-child(): 

p:first-child {
  color: blue;
}

a:hover {
  text-decoration: underline;
}

Pseudo-Elements

Pseudo-elements allow you to style specific parts of an element. For example, ::before and ::after: 

h1::after {
  content: " ★";
  color: gold;
}

Chapter Summary

  • Use transition for smooth property changes
  • Use @keyframes and animation for advanced motions
  • Pseudo-classes represent element states like :hover or :first-child
  • Pseudo-elements target part of elements, e.g., ::before or ::after
  • Combining transitions, animations, and pseudo-selectors allows rich UI effects
