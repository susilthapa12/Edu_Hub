PHP Control Structures

Control structures let you control the flow of your code: conditions and loops.

1. if, else, elseif

<?php
$age = 20;

if ($age >= 18) {
    echo "You are an adult.";
} elseif ($age >= 13) {
    echo "You are a teenager.";
} else {
    echo "You are a child.";
}
?>

2. switch Statement

<?php
$day = "Tuesday";

switch($day) {
    case "Monday":
        echo "Today is Monday";
        break;
    case "Tuesday":
        echo "Today is Tuesday";
        break;
    default:
        echo "Another day";
}
?>

3. for Loop

<?php
for ($i = 1; $i <= 5; $i++) {
    echo "Number: $i <br>";
}
?>

4. while Loop

<?php
$i = 1;
while ($i <= 5) {
    echo "Number: $i <br>";
    $i++;
}
?>

5. do-while Loop

<?php
$i = 1;
do {
    echo "Number: $i <br>";
    $i++;
} while ($i <= 5);
?>

6. foreach Loop

<?php
$fruits = array("Apple", "Banana", "Cherry");

foreach ($fruits as $fruit) {
    echo $fruit . "<br>";
}
?>
