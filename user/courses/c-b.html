C Programming Installation & Setup

To run C programs, you need a compiler. Here’s how to install C on different platforms:

1. Windows (Using MinGW)

  1. Download MinGW from https://www.mingw-w64.org/.
  2. Install MinGW and select gcc during setup.
  3. Add MinGW bin folder to your system PATH environment variable.
  4. Verify installation by opening Command Prompt and typing:
gcc --version

2. Linux (Using GCC)

Most Linux distributions have GCC installed. To install or update:

sudo apt update        # Debian/Ubuntu
sudo apt install build-essential

gcc --version

3. MacOS (Using Xcode Command Line Tools)

Install GCC via Xcode:

xcode-select --install

gcc --version

4. Writing and Compiling a Program

Create a C file hello.c:

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

Compile and run:

gcc hello.c -o hello   # Compile
./hello                 # Run (Linux/Mac)
hello.exe               # Run (Windows)
