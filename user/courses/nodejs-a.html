Node.js Modules

Modules are reusable pieces of code in Node.js. Node has built-in modules and allows creating custom modules.

// Using built-in 'fs' module
const fs = require('fs');
fs.writeFileSync('message.txt', 'Hello Node.js Modules!');
// Creating a custom module (math.js)
exports.add = (a, b) => a + b;
exports.sub = (a, b) => a - b;

// Using the custom module
const math = require('./math');
console.log(math.add(5, 3)); // 8

Event Loop

The Event Loop allows Node.js to perform non-blocking I/O operations, even though JavaScript is single-threaded.

console.log('Start');

setTimeout(() => {
  console.log('Inside Timeout');
}, 0);

console.log('End');

// Output:
// Start
// End
// Inside Timeout

File System (fs)

Node.js allows reading and writing files using the built-in fs module.

// Asynchronous file read
const fs = require('fs');
fs.readFile('message.txt', 'utf8', (err, data) => {
  if (err) throw err;
  console.log(data);
});

// Synchronous file read
const data = fs.readFileSync('message.txt', 'utf8');
console.log(data);
