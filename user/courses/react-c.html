Understanding JSX

JSX is a syntax extension for JavaScript that looks like HTML and allows writing UI components easily.

const element = <h1>Hello, React!</h1>;

JSX expressions can include JavaScript inside curly braces:

const name = 'Mohan';
const greeting = <h1>Hello, {'{'}name{'}'}!</h1>;

Props in React

Props are inputs to components, allowing data to be passed from parent to child.

function Welcome(props) {
  return <h1>Hello, {'{'}props.name{'}'}!</h1>;
}

<Welcome name="Mohan" />

State in React

State is internal data managed by a component. Changing state triggers a re-render.

function Counter() {
  const [count, setCount] = React.useState(0);
  return (
    <div>
      <p>Clicked {'{'}count{'}'} times</p>
      <button onClick={() => setCount(count + 1)}>Click me</button>
    </div>
  );
}
Show Advanced Example 
function ToggleMessage() {
  const [visible, setVisible] = React.useState(false);
  return (
    <div>
      <button onClick={() => setVisible(!visible)}>Toggle Message</button>
      {visible && <p>Hello! Message is visible.</p>}
    </div>
  );
}

Summary

  • JSX allows mixing HTML-like syntax with JavaScript.
  • Props allow passing data to components.
  • State allows components to manage internal data and respond to user interactions.
