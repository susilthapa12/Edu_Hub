AngularJS Data Binding

AngularJS provides two-way data binding, meaning changes in the model update the view, and changes in the view update the model:

<div ng-app="">
  <p>Name: <input type="text" ng-model="name"></p>
  <p>Hello {{name}}!</p>
</div>

Any changes in the input field automatically reflect in the paragraph below.

AngularJS Event Handling

AngularJS supports event directives like ng-click to call functions when events occur:

<div ng-app="myApp" ng-controller="myCtrl">
  <button ng-click="increment()">Click Me</button>
  <p>Counter: {{counter}}</p>
</div>

<script>
var app = angular.module('myApp', []);
app.controller('myCtrl', function($scope) {
  $scope.counter = 0;
  $scope.increment = function() {
    $scope.counter++;
  };
});
</script>

Clicking the button increases the counter dynamically using AngularJS bindings.

Summary

  • AngularJS provides two-way data binding with ng-model.
  • Changes in model update view and vice versa.
  • Event directives like ng-click trigger controller functions.
  • Controllers manage model data and handle events.
