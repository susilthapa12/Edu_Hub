Python Strings

Strings are sequences of characters enclosed in quotes. Single (' ') or double (" ") quotes can be used.

# Examples
name = "Alice"
greeting = 'Hello Python!'
print(name)
print(greeting)

String Operations

# Concatenation
first = "Hello"
last = "World"
full = first + " " + last
print(full)  # Hello World

# Repetition
laugh = "Ha" * 3
print(laugh)  # HaHaHa

Python Input

Input lets the user type data. All input is treated as a string unless converted.

Try these examples in a local Python environment or online Python compiler.

Python Comments

Comments help explain your code. Python ignores them during execution.

# Single-line comment
print("Hello World")  # This prints Hello World

'''
Multi-line comment:
You can use triple quotes to explain multiple lines.
'''
