NumPy Indexing

You can access elements of arrays using indices.

import numpy as np

arr = np.array([10, 20, 30, 40, 50])
print(arr[0])  # First element
print(arr[-1]) # Last element

NumPy Slicing

Slice arrays to access multiple elements.

import numpy as np

arr = np.array([10, 20, 30, 40, 50])
print(arr[1:4])  # Elements at index 1,2,3
print(arr[:3])   # First three elements
print(arr[::2])  # Every second element

Matrix Operations

NumPy supports matrix addition, multiplication, and transpose.

import numpy as np

A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])

# Addition
print(A + B)

# Matrix multiplication
print(A @ B)

# Transpose
print(A.T)

Boolean Indexing

Filter arrays using conditions.

import numpy as np

arr = np.array([10, 20, 30, 40, 50])
print(arr[arr > 25])  # [30 40 50]
