CSS Selectors

What are CSS Selectors?

CSS selectors are patterns used to select the HTML elements you want to style.

Example: Element Selector

This selector targets all <p> elements on the page:

p {
  color: red;
  font-size: 18px;
}

Example: Class Selector

This selector targets all elements with class highlight: 

.highlight {
  background-color: yellow;
  font-weight: bold;
}

Example: ID Selector

This selector targets the element with ID main-title: 

#main-title {
  color: blue;
  text-align: center;
}

Example: Multiple Selectors

You can style multiple elements at once by separating them with commas: 

h1, h2, h3 {
  font-family: Arial, sans-serif;
  color: green;
}

Chapter Summary

  • CSS selectors are used to select HTML elements
  • Element selectors target HTML tags
  • Class selectors target elements with a specific class
  • ID selectors target elements with a specific ID
  • Multiple selectors can be styled together using commas
