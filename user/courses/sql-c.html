CREATE TABLE

The CREATE TABLE statement is used to create a new table in the database.

CREATE TABLE Customers (
    CustomerID int,
    CustomerName varchar(255),
    ContactName varchar(255),
    Country varchar(255)
);

This creates a table named Customers with four columns.

ALTER TABLE

The ALTER TABLE statement is used to modify an existing table.

ALTER TABLE Customers
ADD Email varchar(255);

This query adds a new column Email to the Customers table.

DROP TABLE

The DROP TABLE statement is used to delete a table from the database.

DROP TABLE Customers;

This query permanently deletes the Customers table and all its data.
