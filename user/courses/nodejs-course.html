Node.js Introduction

In this tutorial, you'll learn:

  • How to install and run Node.js
  • Core concepts like modules and the event loop
  • How to build web servers and APIs
  • Working with databases and files
  • Deploying Node.js applications

Node.js is a free, open-source JavaScript runtime that runs on Windows, Mac, Linux, and more. It allows executing JavaScript code outside a browser, enabling server-side development.

Example: Print a Message

console.log('Hello from Node.js!');

Asynchronous Programming

Node.js uses asynchronous (non-blocking) programming, allowing multiple tasks simultaneously.

// Load filesystem module
const fs = require('fs');

// Read file asynchronously
fs.readFile('myfile.txt', 'utf8', (err, data) => {
  if (err) { console.error('Error reading file: ' + err); return; }
  console.log('File content: ' + data);
});

console.log('Reading file... (this runs first!)');

Example: Simple Web Server

const http = require('http');

http.createServer((req, res) => {
  res.writeHead(200, {'Content-Type': 'text/plain'});
  res.end('Hello World!');
}).listen(8080);

Node.js Files

Node.js files have .js extension. They run on the server using the node command:

node app.js

Node.js Versions & LTS

Node.js releases a major version every 6 months. Use the LTS (Long Term Support) version for production projects for stability.
