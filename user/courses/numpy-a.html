What is NumPy?

NumPy is a Python library used for working with arrays, linear algebra, Fourier transforms, and matrices.

Created in 2005 by Travis Oliphant, NumPy is open-source and freely usable.

NumPy stands for Numerical Python.

Why Use NumPy?

Python lists are slow for numerical operations. NumPy provides ndarray, which is up to 50x faster than lists.

Arrays are stored contiguously in memory, enabling fast access and operations.

NumPy is widely used in data science, where speed and efficiency are crucial.

Example: Creating a NumPy Array

import numpy as np

# Create a 1D array
arr = np.array([1, 2, 3, 4, 5])
print(arr)
