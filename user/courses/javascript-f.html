JavaScript Loops

Loops let you run the same block of code multiple times. Instead of repeating code, use loops to repeat it automatically.

The for Loop

use when iterations known 
<!DOCTYPE html>
<html>
<body>
  <p id="demo"></p>
  <script>
    let text = "";
    for (let i = 1; i <= 5; i++) {
      text += "Number " + i + "<br>";
    }
    document.getElementById("demo").innerHTML = text;
  </script>
</body>
</html>

The while Loop

Runs while the condition is true.

let i = 1;
while (i <= 5) {
  document.write("Number " + i + "<br>");
  i++;
}

The do...while Loop

Runs the block at least once, then repeats while condition is true.

let j = 6;
do {
  document.write("Number is " + j + "<br>");
  j++;
} while (j <= 5);

The for...of Loop

Used to iterate arrays, strings and other iterable objects.

let colors = ["Red", "Green", "Blue"];
for (let c of colors) {
  document.write(c + "<br>");
}

The for...in Loop

Used to iterate over object properties.

let person = {name: "John", age: 25, city: "New York"};
for (let key in person) {
  document.write(key + ": " + person[key] + "<br>");
}
