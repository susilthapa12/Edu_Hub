JavaScript Loops

Loops let you run the same block of code multiple times. Instead of writing code again and again, you use a loop to repeat it automatically.

The for Loop

The for loop is used when you know how many times you want to run the code. 

<!DOCTYPE html>
<html>
<body>
  <p id="demo"></p>
  <script>
    let text = "";
    for (let i = 1; i <= 5; i++) {
      text += "Number " + i + "<br>";
    }
    document.getElementById("demo").innerHTML = text;
  </script>
</body>
</html>

The while Loop

The while loop runs as long as the condition is true. 

let i = 1;
while (i <= 5) {
  document.write("Number " + i + "<br>");
  i++;
}

The do...while Loop

The do...while loop runs the code at least once, even if the condition is false. 

let j = 6;
do {
  document.write("Number is " + j + "<br>");
  j++;
} while (j <= 5);

The for...of Loop

The for...of loop is used to loop through arrays, strings, or lists. 

let colors = ["Red", "Green", "Blue"];
for (let c of colors) {
  document.write(c + "<br>");
}

The for...in Loop

The for...in loop is used to loop through object properties. 

let person = {name: "John", age: 25, city: "New York"};
for (let key in person) {
  document.write(key + ": " + person[key] + "<br>");
}
