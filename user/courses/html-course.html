HTML Introduction

HTML (HyperText Markup Language) is the standard language for creating webpages. It structures content for browsers to display.

HTML Basic Example

Every HTML document starts with a <!DOCTYPE html> declaration and contains html, head, and body elements.

<!DOCTYPE html>
<html>
<head>
  <title>My First Page</title>
</head>
<body>
  <h1>Hello World!</h1>
  <p>This is my first webpage.</p>
</body>
</html>

HTML Elements

HTML elements are building blocks of webpages. Examples include headings, paragraphs, links, and images.

<h1>Main Heading</h1>
<p>A paragraph of text.</p>
<a href="https://example.com">Visit Example</a>
<img src="photo.jpg" alt="Sample Photo">

HTML Attributes

Attributes provide additional information about elements, usually in the form of name="value".

<a href="https://google.com" target="_blank">Open Google</a>
