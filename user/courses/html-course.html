HTML Introduction

HTML (HyperText Markup Language) structures content for webpages, providing headings, paragraphs, links, images, and more.

HTML Basic Example

<!DOCTYPE html>
<html>
<head>
  <title>My First Page</title>
</head>
<body>
  <h1>Hello World!</h1>
  <p>This is my first webpage.</p>
</body>
</html>

HTML Elements

<h1>Main Heading</h1>
<p>A paragraph of text.</p>
<a href="https://example.com">Visit Example</a>
<img src="photo.jpg" alt="Sample Photo">

HTML Attributes

<a href="https://google.com" target="_blank">Open Google</a>
