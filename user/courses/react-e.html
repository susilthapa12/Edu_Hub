Rendering Lists in React

React can render arrays of data using the map() method.

const numbers = [1, 2, 3, 4, 5];
const listItems = numbers.map((number) =>
  <li key={number}>{number}</li>
);

function NumberList() {
  return <ul>{listItems}</ul>;
}

Notice the key prop added to each list item. Keys help React identify which items changed, are added, or removed.

Keys in React

Keys must be unique among siblings. They are essential for performance and to maintain component identity.

const todos = [
  {id: 1, text: 'Learn React'},
  {id: 2, text: 'Build a project'},
];

function TodoList() {
  return (
    <ul>
      {todos.map(todo => (
        <li key={todo.id}>{todo.text}</li>
      ))}
    </ul>
  );
}

Dynamic Lists with State

You can render dynamic lists using state and event handlers:

function ShoppingList() {
  const [items, setItems] = React.useState(['Apples', 'Bananas', 'Oranges']);

  return (
    <ul>
      {items.map((item, index) => (
        <li key={index}>{item}</li>
      ))}
    </ul>
  );
}

Summary

  • Use map() to render arrays in React.
  • Always provide a unique key prop to each list item.
  • Keys help React efficiently update and maintain components.
  • Lists can be dynamic when combined with state and event handling.
