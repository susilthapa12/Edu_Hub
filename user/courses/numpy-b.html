NumPy Arrays

The ndarray is the core array object in NumPy. It allows for fast and vectorized operations.

import numpy as np

# Create a 2D array
arr = np.array([[1, 2, 3], [4, 5, 6]])
print(arr)

Array Operations

You can perform arithmetic and statistical operations directly on arrays.

import numpy as np

a = np.array([1, 2, 3])
b = np.array([4, 5, 6])

# Addition
print(a + b)  # [5 7 9]

# Multiplication
print(a * b)  # [4 10 18]

# Mean
print(np.mean(a))  # 2.0

Reshaping Arrays

NumPy arrays can be reshaped without changing the underlying data.

import numpy as np

arr = np.arange(12)
print(arr.reshape(3, 4))
