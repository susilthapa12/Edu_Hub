Creating a Database

Use CREATE DATABASE to create a new database:

-- Create a database named 'MySchool'
CREATE DATABASE MySchool;

To use the database:

USE MySchool;

Creating a Table

Use CREATE TABLE to define a new table with columns:

CREATE TABLE Students (
    StudentID INT AUTO_INCREMENT PRIMARY KEY,
    FirstName VARCHAR(50),
    LastName VARCHAR(50),
    Age INT,
    Grade VARCHAR(10)
);

Inserting Data

Use INSERT INTO to add records:

INSERT INTO Students (FirstName, LastName, Age, Grade)
VALUES ('John', 'Doe', 15, '10th');

INSERT INTO Students (FirstName, LastName, Age, Grade)
VALUES ('Jane', 'Smith', 16, '11th');

Querying Data

Use SELECT to view records:

-- Select all columns
SELECT * FROM Students;

-- Select specific columns
SELECT FirstName, LastName FROM Students;

Updating Data

Use UPDATE to modify records:

UPDATE Students
SET Grade = '11th'
WHERE StudentID = 1;

Deleting Data

Use DELETE to remove records:

DELETE FROM Students
WHERE StudentID = 2;
