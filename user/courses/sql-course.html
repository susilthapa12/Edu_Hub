Introduction to SQL

SQL stands for Structured Query Language and is the standard language for accessing and manipulating databases.

SQL became a standard of the American National Standards Institute (ANSI) in 1986, and of the International Organization for Standardization (ISO) in 1987.

What Can SQL Do?

  • Execute queries against a database
  • Retrieve data from a database
  • Insert records in a database
  • Update records in a database
  • Delete records from a database
  • Create new databases
  • Create new tables in a database
  • Create stored procedures and views
  • Set permissions on tables, procedures, and views

Note: Different SQL versions exist, but all support the major commands (SELECT, UPDATE, DELETE, INSERT, WHERE).

Using SQL in Your Website

  • An RDBMS database program (e.g., MS Access, SQL Server, MySQL)
  • A server-side scripting language like PHP or ASP
  • SQL to query and manipulate data
  • HTML/CSS to style your pages

Relational Database Management System (RDBMS)

RDBMS is the foundation of SQL and modern database systems such as MS SQL Server, IBM DB2, Oracle, MySQL, and Microsoft Access.

Data is stored in tables consisting of rows and columns.

SELECT * FROM Customers;

Each row is a record, and each column is a field storing specific data.

Example table fields: CustomerID, CustomerName, ContactName, Address, City, PostalCode, Country.
