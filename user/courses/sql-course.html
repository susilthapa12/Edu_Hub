Introduction to SQL

SQL stands for Structured Query Language and is the standard language for accessing and manipulating databases.

SQL became a standard of ANSI in 1986 and ISO in 1987.

What Can SQL Do?

  • Execute queries against a database
  • Retrieve data from a database
  • Insert, update, delete records
  • Create new databases and tables
  • Create stored procedures and views
  • Set permissions on tables and procedures

Note: All major SQL versions support SELECT, UPDATE, DELETE, INSERT, WHERE commands.

Using SQL in Your Website

  • RDBMS (MS Access, SQL Server, MySQL)
  • Server-side scripting (PHP, ASP)
  • SQL for data queries
  • HTML/CSS for page display

Relational Database Management System (RDBMS)

RDBMS stores data in tables with rows and columns.

SELECT * FROM Customers;

Each row is a record; each column stores a field.

Example columns: CustomerID, CustomerName, ContactName, Address, City, PostalCode, Country.
