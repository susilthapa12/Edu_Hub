INSERT Statement

The INSERT statement is used to add new records into a table.

INSERT INTO Customers (CustomerName, ContactName, Country)
VALUES ('Maria Anders', 'Maria', 'Germany');

This query adds a new customer to the Customers table.

UPDATE Statement

The UPDATE statement modifies existing records in a table.

UPDATE Customers
SET ContactName = 'Maria Smith', Country = 'Germany'
WHERE CustomerID = 1;

This query updates the customer with CustomerID = 1.

DELETE Statement

The DELETE statement removes records from a table.

DELETE FROM Customers
WHERE CustomerID = 1;

This query deletes the customer with CustomerID = 1 from the table.
