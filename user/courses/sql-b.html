INSERT Statement

Add new records to a table:

INSERT INTO Customers (CustomerName, ContactName, Country)
VALUES ('Maria Anders', 'Maria', 'Germany');

UPDATE Statement

Modify existing records:

UPDATE Customers
SET ContactName = 'Maria Smith', Country = 'Germany'
WHERE CustomerID = 1;

DELETE Statement

Remove records from a table:

DELETE FROM Customers
WHERE CustomerID = 1;
