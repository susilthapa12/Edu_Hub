PHP Functions

Functions are blocks of code designed to perform specific tasks. They help in reusing code.

1. Defining a Function

<?php
function sayHello() {
    echo "Hello, World!";
}

sayHello(); // Call the function
?>

2. Function with Parameters

<?php
function greet($name) {
    echo "Hello, $name!";
}

greet("Alice");
greet("Bob");
?>

3. Function with Return Value

<?php
function add($a, $b) {
    return $a + $b;
}

$result = add(5, 3);
echo "Sum: $result";
?>

4. Default Parameters

<?php
function greetUser($name = "Guest") {
    echo "Welcome, $name!";
}

greetUser();        // Uses default: Guest
greetUser("Alice"); // Overrides default
?>

5. Variable Scope

Variables inside a function are local by default. Use global to access global variables.

<?php
$x = 10;

function testScope() {
    global $x;
    echo $x; // Accesses global $x
}

testScope();
?>

6. Anonymous Functions / Closures

<?php
$sayHi = function($name) {
    echo "Hi, $name!";
};

$sayHi("Alice");
?>
