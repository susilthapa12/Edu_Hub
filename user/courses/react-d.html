React Events

React handles events similarly to HTML, but with JSX syntax. Use camelCase for event names.

function ClickButton() {
  function handleClick() {
    alert('Button clicked!');
  }
  return <button onClick={handleClick}>Click Me</button>;
}

React automatically passes the event object as an argument to event handlers:

function InputChange() {
  function handleChange(event) {
    console.log(event.target.value);
  }
  return <input onChange={handleChange} />;
}

Conditional Rendering

You can render elements conditionally using JavaScript expressions inside JSX.

function Greeting(props) {
  const isLoggedIn = props.isLoggedIn;
  return (
    <div>
      {isLoggedIn ? <h1>Welcome back!</h1> : <h1>Please sign in.</h1>}
    </div>
  );
}

Or use short-circuit evaluation for simpler conditions:

function Mailbox(props) {
  const unreadMessages = props.unreadMessages;
  return (
    <div>
      {unreadMessages.length > 0 && <h2>You have {unreadMessages.length} unread messages.</h2>}
    </div>
  );
}

Summary

  • React events use camelCase and JSX syntax.
  • Event handlers can access event objects directly.
  • Conditional rendering allows dynamic content display using ternary operators or short-circuit evaluation.
